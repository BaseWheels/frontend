# Components

Folder ini berisi komponen React yang bisa digunakan kembali (reusable components).

## Struktur:

### `ui/`
Komponen UI dasar yang bisa dipakai di berbagai halaman:
- Button
- Input
- Card
- Modal
- dll

### `shared/`
Komponen shared yang dipakai di banyak halaman:
- Header
- Footer
- Navigation
- Sidebar
- dll

## Contoh penggunaan:
```jsx
import { Button } from '@/components/ui/Button'
import { Header } from '@/components/shared/Header'
```
