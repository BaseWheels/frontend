# Lib

Folder ini berisi utilities, helpers, dan fungsi-fungsi pembantu.

## Isi folder:
- **utils.js** - Fungsi-fungsi utility umum
- **constants.js** - Konstanta yang dipakai di seluruh aplikasi
- **api.js** - API client / fetch wrapper
- **validation.js** - Fungsi validasi
- dll

## Contoh penggunaan:
```jsx
import { formatCurrency, formatDate } from '@/lib/utils'
import { API_BASE_URL } from '@/lib/constants'
```
